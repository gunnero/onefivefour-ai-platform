# Architecture

OneFiveFour AI Media Platform is separate from any CMS or public website.

It provides AI services to publishing systems.

## First integration

Razbudise.mk will use OneFiveFour AI Media Platform as its AI workforce.

## Communication

Razbudise communicates with the AI Platform through:

- REST API
- Webhooks
- Later: queues or event bus

## Ownership rule

The AI Platform creates recommendations, drafts, research, SEO metadata, and media assets.

The CMS owns published content.
