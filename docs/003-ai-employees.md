# AI Employees

AI Employees are specialized AI agents represented like team members.

Each AI Employee has:

- Full name
- Avatar
- Role
- Department
- Bio
- Skills
- Permissions
- Tools
- Prompt profile
- Memory
- Tasks
- Logs
- Performance metrics
